**Как осуществляется перевод**
Сама игра не поддерживает русские шрифты. Для перевода был выбран метод поиска не используемых символов и замены их на русские. 
То есть как это происходит:
- Вы пишите текст на русском 
- Прогоняете файл перевода через приложеный софт
- Софт заменяет к примеру "e" на символ "a" который игра распознаёт
- Игра в свою очередь распознаёт символ и вставляет нужную букву, но ввиду того что файлы описания шрифтов заранее отредактированы то отрисовывается нужный русский символ.

**Как работает софт**
- file replacer<sup>1</sup> заменяет нужные символы из входного файла и записывает результат в выходной
- string replacer заменяет нужные символы из входной строки и записывает результат в окно ниже

Все исходные коды есть рядом с софтом.
<sup>1</sup>-нужная платформа.

**Папка fonts**
Папка содержит файлы которые нужны для корректной подстановки русских символов.
- psd файлы фотошопа
- normal файлы которые нужно закинуть в корень кэша (obb файла)
- original оригинальные файлы
- colored каждый шрифт разного цвета. Для поиска ошибок и мало ли для чего

**Что нужно переводить**
- obb/HudText_US.mlt обычный текстовик (часть переведена)
- obb/English.lng бинарный файл с вкраплениями текста (лучше открывать в notepad++ или любом hex-редакторе)
- obb/*.pob файлы построек. Бинарные файлы. Всё что нужно находится в пределах 10 символов перевода каретки(\LF).  (лучше открывать в notepad++ или любом hex-редакторе)
- apk/lib/< arch >/libCarbonAndroid.so библиотека с вкраплениями текста (лучше открывать в notepad++ или любом hex-редакторе). НЕЛЬЗЯ ПРЕВЫШАТЬ ДЛИНУ ОРИГИНАЛЬНОГО ВЫРАЖЕНИЯ, недобор символов можно добивать нулями.
Для armeabi-v7a основной массив текста начинается по сдвигу 0x82fc1b

Файлы лучше брать с новых версий так что выкладывать не буду.
